Thank you for your interest in contributing to AirLakeFlow.

Please follow these guidelines to contribute:

1. Fork the repository and create a branch for your change.
2. Follow the code style (Black) and linting (Ruff).
3. Write tests for new features or bug fixes. Use pytest and CliRunner for CLI tests.
4. Run the test suite locally: pytest -q
5. Open a pull request describing your changes and link related issues.

CI will run linters, tests, and basic security checks on every PR.
